# المخزن المؤقت الدائري (Circular Buffer) - تطبيق بلغة C

## وصف المشروع
هذا برنامج تعليمي بسيط يوضح آلية عمل **المخزن المؤقت الدائري** (Circular Buffer) أو ما يُسمى بـ **FIFO queue** بحجم ثابت.  
يقوم البرنامج بـ:
- قراءة اسم المستخدم من لوحة المفاتيح.
- إضافة اللاحقة `"CE-ESY"` إلى الاسم.
- تخزين كل حرف من النتيجة في مخزن دائري سعته 10 أحرف.
- قراءة الأحرف المخزنة وعرضها بنفس الترتيب (FIFO).
- إظهار رسائل خطأ عند overflow أو underflow.

## متطلبات التشغيل
- مترجم لغة C (مثل GCC).
- نظام تشغيل: Windows / Linux / macOS.

## كيفية البناء والتشغيل
1. احفظ الكود المصدري في ملف مثلاً `circular_buffer.c`.
2. افتح الطرفية (Terminal / CMD) في مجلد الملف.
3. قم بترجمة البرنامج:
   ```bash
   gcc circular_buffer.c -o circular_buffer
